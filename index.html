Projects

Projects

Online Form Automation

Created an application in python and C# to automate the process of creating check boxes, labels, field sets and text boxes for online forms. Utilizing a csv file that stores all the contents of the forms. The program generates the html code to produce the checkbox, text box, labels, and field sets. The html code is written to a file and stored within the directory. This program reduces the time to create a custom form by up to 2 hours.

Televox Migration Project

Televox Migration Project consisted of a group of 10 interns, transferring data from the old televox system to the newly developed system. This process entailed migrating content, search engine optimization, and quality assurance for these sites. This process occured for approximately 250 client websites.

Cash Close

Cash Close is an application developed in Python. It is used to aid a cashier to balance the cash registers at a grocery store for cash reconciliation. The application receives input about the quantity of different types of coins, bills and coupons. Through the inputs, it determines the most efficient way to balance the cash register.
